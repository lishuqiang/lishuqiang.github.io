Hexo常用命令

简写

hexo n post “文件名” == hexo new post “文件名”#新建md文章
hexo s == hexo server#启动本地预览
hexo d == hexo deploy#页面部署
hexo g == hexo generate#生成
hexo p == hexo publish#草稿变成文章
hexo d -g == hexo d + hexo d#页面部署并生成

服务器

hexo server#启动本地预览并监视文件变动并自动更新
hexo server -s#静态模式
hexo server -p 5000#更改端口5000可改
hexo server -i 192.168.1.1#自定义ip
hexo clean#清除缓存 网页正常情况下可以忽略
hexo g#生成静态网页
hexo d#开始部署

监视文件变动

hexo generate#使用Hexo生成静态文件快速并且简单
hexogenerate –watch#监视文件变动

完成后部署

hexo generate –deploy
hexo deplay –generate

hexo deploy -g
hexo server -g
hexo d -g
hexo g -d

草稿

hexo publish [layout]

模版

#新建文章 hexo new page “pageName”**#新建页面

hexo generate#生成静态页面至public目录

hexo server#开启预览访问端口（默认端口为4000，’ctrl+c’关闭server）
