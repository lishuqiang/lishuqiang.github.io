int m_iAge;如果是成员变量，均以m_开始
int iNumber；如果是普通变量，取与实际意义相关的名称，并且添加类型的首字母，而且首字母大写

常量

定义：在程序中不可改变的量
常量可以分为三大类：
1、数值型常量（包括整型常量和实型常量）
2、字符型常量
3、符号常量

整型常量

定义：就是直接还用的整型常数，例如：123、-6209。8等。
常量可以分为四类，分别是：
1、长整型  2、短整型  3、符号整型  4、无符号整型
在编写整型常量时，可以在常量的后面加上符号L或者U进行修饰。L表示该常量是长整型，U表示该常量为无符号整型，例如：
LongNum= 1000L; 　　　　　　 UnsignLongNum=500U;
L和U可以用大写，也可以使用小写。
整型常量可以通过3种形式表达，分别是八进制形式、十进制形式、十六进制形式。
1、八进制形式（0~7之间）
例如：
OctalNumber1=0123;（必须是4位数，且不包含大于7的数
2、十进制整数
不需要在其前面添加前缀的。
例如：
AlgorismNumber1=123
3、十六进制整数
常量前面使用0×作为前缀。十六进制中包含数字0~9，和字母A~F。
例如：
HexNumber1=0×123；
其中字母可以是大写，也可以是小写

实型常量

实型也称为浮点型，由整数部分和小数部分组成，并用十进制的小数点进行分隔。表示实数的方式有两种，分别是：科学计数方式和指数方式。
1、科学计数方式
例如：
SciNum1=123.45
SciNum2=0.5458
2、指数方式
使用字母e或E进行指数表示
例如：
SciNum1=1.2345e2所表示的值为123.45
SciNum2=5.458e-1所表示的值为0.5458
在编写实型常量时，可以在常量的后面加上符号F或者L进行修饰。F代表float，单精度类型，L代表long，长双精度类型。
例如：
FloatNum= 1.2345e2F单精度类型
LongDoubleNum=5.458E-1L长双精度类型

字符型常量

字符型常量可以分成两种，分别是字符常量和字符串常量
1、字符常量
例如：’A’、’#’
2、字符串常量
例如：“Hello world”、“welcome”
字符串“welcome”在内存中的存储形式为：
welcpme\0
